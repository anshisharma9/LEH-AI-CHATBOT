# LEH AI Chatbot

AI Powered Educational Assistant for Learning Education Hub.

## Features

* AI Career Guidance
* Course Information
* Teacher Information
* Class Timings
* Notices & Updates
* FAQ Support
* Admin Dashboard

## Tech Stack

### Frontend

* React.js
* Vite
* Axios

### Backend

* FastAPI
* SQLAlchemy
* JWT Authentication

### Database

* MySQL

### AI

* Google Gemini API

## Installation

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```
